2. Write Path - Encrypt on Message Send

1 User Sends Message INPUT

Content: "Hi, my email is john@example.com"

2 detect_sensitive_data() DETECT
def detect_sensitive_data(content):
  # Scan for email, phone, card, SSN
  return {
    "has_pii": True,
    "types": ["email"]
  }
3 encrypt_sensitive_data() + mask_sensitive_data() ENCRYPT + MASK
# Two operations happen:
encrypted = encrypt("john@example.com") → "gAAA...ciphertext"
masked = mask("john@example.com") → "j***@example.com"
4 Database Write SAVE
FieldValue
content"j***@example.com" (masked)
encrypted_content"gAAA...encrypted"
has_piiTRUE
